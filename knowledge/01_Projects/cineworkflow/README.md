# CineWorkflow Project

**Status:** MVP Live (Active)  
**Started:** 2026-02-24  
**Goal:** $1K+/month revenue from AI video prompt marketplace

## Overview
CineWorkflow is a curated prompt marketplace for AI video generation tools (Runway, Pika, Kling, Luma, Sora). Users get professional prompts that "just work" instead of trial-and-error.

**Live URL:** https://cineworkflo.com

## Success Metrics
- **Revenue:** $1K+/month target
- **Users:** Email list growth, conversion to Pro
- **Content:** 90 prompts (25 free, 65 Pro)
- **Tools Supported:** Runway, Pika, Kling, Luma, Sora, CinemaStudio (planned)
